Callan and Ashley's **NWEN304 Group Project**

## App Design

This will be a basic marketplace app for coffee that the user can log into and add items into their cart.

## Features
* Shop page
* User login
  * Authenticated Login
  * Encrypted
  * Facebook / Google Login
* Admin Login
  * Add new items
  * Remove items
  * Edit items
